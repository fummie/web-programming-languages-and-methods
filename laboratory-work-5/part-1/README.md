## Вариант задачи
Вычислить:
$$
  \frac{sin(a)-b}{|b|+cos(b^2)}
$$

## Инициализация проекта
1. Установка зависимостей
```bash
bundle install
rspec --init
```
2. Запуск тестов
```bash
bundle binstubs rspec-core
./bin/rspec
```
3. Запуск проекта
```bash
ruby ./lib/main.rb
```
